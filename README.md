# C++ Game Project

A simple C++ project for game programming.

## Getting Started

### Prerequisites

- A C++ compiler (like g++)

### Building the Project

To build the project, run the `make` command:

```bash
make
```

This will compile the source code and create an executable named `game`.

### Running the Application

To run the application, execute the following command:

```bash
./game
```

### Cleaning Up

To remove the compiled files, run the `make clean` command:

```bash
make clean
```
